# Projet Lirann - Le Guide du Cafe

## Description
Site web statique presentant les differents types de cafes du monde entier avec leurs recettes de preparation. Design moderne, theme sombre avec accents ambre/cafe.

## Structure
- `index.html` â€” Fichier unique contenant HTML + CSS + JS (pas de framework, pas de dependances)

## Contenu
25 types de cafes repartis en 5 categories :
- **Espresso** : Espresso, Ristretto, Lungo, Americano
- **A base de lait** : Cappuccino, Latte, Flat White, Macchiato, Cortado, Latte Macchiato, Mocha, Cafe au Lait
- **Filtre & Infusion** : V60, Chemex, French Press, AeroPress, Moka (Bialetti)
- **Cafe froid** : Cold Brew, Iced Latte, Affogato, Espresso Tonic
- **Du monde** : Cafe Turc, Cafe Vietnamien, Frappe Grec, Irish Coffee

## Fonctionnalites
- Hero plein ecran avec animation
- Filtrage par categorie (barre sticky)
- Grille responsive de cartes avec hover effects
- Modal detaillee par cafe : description, ingredients, etapes numerotees, infos (cafeine, temperature, temps)
- Donnees stockees en JS (pas de backend)
- Responsive mobile
- Fonts : Playfair Display + Inter (Google Fonts)
